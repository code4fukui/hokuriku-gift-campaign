# 北陸観光 インバウンド統計データ可視化

<https://welcome-hokuriku.jp>で回答されたアンケートデータをオープンデータとして公開しています。このデータは[北陸インバウンド観光DX・データコンソーシアム](https://github.com/hokuriku-inbound-kanko)によって管理・公開されています。

## デモ
[北陸インバウンド観光DX オープンデータ可視化ウェブアプリケーション](https://hokuriku-inbound-kanko.github.io/hokuriku-inbound-kanko-visualization/)でグラフとして閲覧可能です。

## データ・API
アンケートデータは`all.csv`ファイルで提供されています。日本語訳は`all_ja.csv`ファイルで確認できます。
